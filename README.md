# Admin Dashboard - Next.js

A modern Next.js admin dashboard with multiple management pages.

## Features

- Dashboard with charts and metrics
- User Management
- License Management
- Ticket & Fine Management
- Document Verification
- Settings

## Installation

```bash
npm install
```

## Running the Project

```bash
npm run dev
```

The application will open at http://localhost:3000

## Technologies Used

- Next.js 14
- React 18
- Recharts (for charts)

## Project Structure

- `app/` - Next.js App Router pages
- `components/` - Reusable React components
- `public/` - Static assets

## Pages

- `/` - Dashboard
- `/user-management` - User List
- `/user-management/user-verification` - User Verification
- `/license-management` - License Management
- `/ticket-management` - Ticket Management
- `/ticket-management/add-ticket` - Add New Ticket
- `/document-verification` - Document Verification
- `/settings` - Settings
- `/settings/edit-personal-info` - Edit Personal Information
